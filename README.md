# AWS DevOps Interview Companion

Build • Learn • Interview • Revise

## Objective

This repository is a production-style AWS DevOps interview preparation project.

It combines:

- Hands-on DevOps implementation
- AWS cloud deployment
- Docker and containerization
- Terraform Infrastructure as Code
- CI/CD automation
- Monitoring and security
- Interview notes
- Scenario-based questions
- Blog-ready documentation

## Primary Target Role

AWS DevOps Engineer

## Secondary Target Roles

- DevOps Engineer
- Cloud Engineer
- Platform Engineer
- AI Platform Engineer
